#  Kids Academy — Web Learning Platform for 10th Grade Students

Kids Academy is an interactive MERN-based learning platform designed specifically for 10th standard students to understand the fundamentals of web development.

The platform introduces HTML, CSS, and JavaScript concepts through structured lessons, guided modules, and interactive UI components tailored for beginner learners.

---

##  Objective

To simplify web development fundamentals for school-level students by providing:

- Step-by-step structured content
- Beginner-friendly explanations
- Interactive UI learning modules
- Practice-driven learning approach

This platform bridges the gap between school-level computer science and real-world web development.

---

##  Learning Modules

-  Introduction to the Internet & Web
-  HTML Basics (Structure, Tags, Forms)
-  CSS Fundamentals (Styling, Layout, Responsive Design)
-  JavaScript Basics (Variables, Functions, DOM)
-  Mini Practice Projects
-  Guided Exercises & Assignments

---

##  System Architecture

Frontend (React + Tailwind)  
↓  
Node.js + Express Backend  
↓  
MongoDB Database  

Built using the MERN stack for scalability and structured content management.
